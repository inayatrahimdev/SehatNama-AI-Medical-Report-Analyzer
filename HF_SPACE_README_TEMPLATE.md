# SehatNama AI - Medical Report Intelligence

Production-oriented healthcare document intelligence app.

- Upload medical reports (`PDF`, `JPG`, `JPEG`, `PNG`)
- OCR + structured extraction
- Handles numeric reports and clinical narrative reports
- English + Urdu structured summary
- Confidence scoring + quality warnings
- JSON export for downstream systems
